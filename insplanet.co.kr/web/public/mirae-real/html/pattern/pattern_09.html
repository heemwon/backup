9-2. 다건신청 UI _ 계속입력형

9-2-1 계속입력형_입력

‘타이틀’ 등록

  • 사업목적은 ⎡한국표준산업분류표⎦의 분류내용을 참고하여 작성해 주시기 바랍니다.
  • 포괄적이거나 추상적인 목적을 입력하시면 안되며 사업의 구체적인 목적을 입력하시기 바랍니다.
* 표시 필수입력사항
사업목적

사업목적 입력 안내

하나의 목적번호에는 하나의 목적만 입력해야 하며, 목적번호는 모두 ‘1’ 로 입력합니다.

예) 입력할 목적이 운수업, 유통업, 각 호에 부대하는 사업인 경우

목적 입력란에 ‘1 운수업’ 입력 후 우측 하단의 [입력] 버튼을 선택하고
다시 입력란에 ‘1 유통업‘ 입력 후 우측 하단의 [입력] 버튼을 선택하고
마지막으로 ‘1 각 호에 부대하는 사업‘ 입력 후 우측 하단의 [입력] 버튼 선택

1.
