IaaS

Object Storage를 이용한 데이터 백업

  • 사용자 데이터 백업

  1. 사용자가 Object storage 상품을 통해 백업 bucket을 생성하고, 접속 URL, access key, secret key 를 발급받는다.

  2. Virtual Server에 백업 서버를 설치하고, 백업 bucket을 백업 디스크(disk pool)로 등록한다.

  3. 백업 환경을 구성하고 agent 백업을 수행한다.

  4. 다른 Region의 Object Storage에 복제한다.

사용 사례

  3. 사용자 데이터 백업

    Virtual Server 또는 Bare Metal Server의 디렉토리와 파일을 백업하고, 그 데이터를 Object Storage에 저장합니다. 백업S/W를 통해 백업 정책을 설정하고, 자동 백업을 수행합니다. 백업 이력를 통해 손쉽게 데이터를 복원할 수 있습니다. 또한 Region간 Object Storage 복제를 제공하므로 재난 상황에서도 백업 데이터를 안전하게 보관하여 복원할 수 있습니다.
