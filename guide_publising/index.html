코드 규칙

주석
시작과 끝의 표시는 START, END(대문자)로 구분합니다.
git으로 완벽하게 되지 않는 형상관리를 대비하여 개발 전달 후 수정사항이 생길 시 "목적, 날짜, 이름, 간단한 설명"을 추가합니다.
간단한 워딩수정이나, 단순 구조일 경우 인라인으로 START, END 구분 없이 작성합니다.
  1. ***일반 작업시
  2. <!-- START : 이름 -->
  3.   .....
  4. <!-- END : 이름 -->
  6. ***개발 전달 후 수정작업시
  7. ***단순 구조
  8. .....<!-- 추가 : 20220509 : 이름 : 설명 -->
  9. ***다중 구조
  10. <!-- START : 수정 : 20220509 : 이름 : 설명 -->
  11. <!-- START : 추가 : 20220509 : 이름 : 설명 -->
  12.   .....
  13. <!-- END : 수정 : 20220509 : 이름 : 설명 -->
  14. <!-- END : 추가 : 20220509 : 이름 : 설명 -->

구분(선언 순서대로 오름차순) 속성 종류
1. 기능 href, src, type
2. 이름 name, id
3. 정보 value, title, alt
4. 옵션 target, tabindex
5. 꾸미기 class
6. 역할 role
7. 속성 aria-controls, aria-describedby, aria-labelledby, aria-popup, aria-modal 등
8. 상태 checked, disabled, selected, required, aria-selected, aria-current, aria-hidden 등
CSS 규칙
따옴표 : content, font-family, image url등 css내부에서 사용되는 따옴표는 작은 따옴표(')를 사용합니다.
  1. .class-name::before {content:'';}
  2. .class-name .child-class {background:url('../../image.jpg') no-repeat center center;}
세미 콜론 : 마지막 선언 속성에도 세미 콜론(;)을 사용합니다.
  1. .class-name::before {content:'';}
공백(1) : 선택자간, 중괄호간, 속성간 한 칸의 공백을 사용합니다.
  1. .class-name .child-class01 .child-class02 {content:''; position:absolute; top:0; left:0;}
공백(2) : css속성과 값사이의 공백은 사용하지 않습니다.
  1. .class-name {content: ''; position: absolute;} (X)
  2. .class-name {content:''; position:absolute;} (O)
들여쓰기(1) : 일반 css 코드 작성시 들여쓰기는 사용하지 않으며, 공통으로 사용되는 속성의 선택자는 직관적으로 확인 할 수 있도록 줄바꿈을 허용합니다.
  1. .class-name01,
  2. .class-name02 {content:''; position:absolute;} (O)
들여쓰기(2) : 미디어쿼리 사용시 1탭 들여쓰기를 허용합니다.
  1. @media screen and (min-width:1920px) {
  2.   .class-name {속성값}
  3. }
SCSS : SCSS의 경우 속성 간 줄 바꿈을 사용, 가상 선택자 사용 시 자식 선택자 이전에 선언합니다.
  1. .class-name {
  2.   display:inline-block;
  3.   width:100px;
  4.   height:100px;
  5.   &::before {
  6.     content:'';
  7.     position:absolute;
  8.   }
  9.   .class-child {
  10.     overflow:hidden;
  11.   }
  12. }
주석 : 공통css 작성 시 종류나, 페이지별로 css작성 시 페이지를 구분하기 위해 주석을 작성한다. (종료 주석은 작성하지 않습니다.)
가독성을 위해 주석 묶음간 1줄의 빈 줄을 사용합니다.
  1. /* button */{
  2. .btn {width:10px; height:10px; background-color:#000;}
  4. /* visual */
  5. .visual {background:url('../../image.jpg') center center; background-size:cover;}
전체선택자(*) : 웹페이지의 성능 저하를 가져올 수 있으므로 사용을 지양합니다.

폼 요소

form
name: 폼 태그 이름
action : 입력 데이터의 전달 위치 지정
method : 입력 데이터의 전달 방식 선택 (get/post)
  1. <form name="폼 태그 이름" action="서버 URI" method="GET 또는 POST">
  2.     <fieldset>
  3.          <legend>입력 양식</legend>
  4.          .....
  5.     </fieldset>
  6. </form>
  8. <form name="폼 태그 이름" action="서버 URI" method="GET 또는 POST" disabled>
  9.     <fieldset>
  10.          <legend>비활성 입력 양식</legend>
  11.          .....
  12.     </fieldset>
  13. </form>

form에 id값과 fieldset에 form 값이 부여된 경우 개별적으로 사용이 가능하다.

  1. <form name="폼 태그 이름" id="formId" action="서버 URI" method="GET 또는 POST">
  2. </form>
  3. <fieldset form="formId">
  4.      <legend>입력 양식</legend>
  5.      .....
  6. </fieldset>
input
title 애트리뷰트를 추가함으로써 스크린 리더 사용자에게 각 요소에 접근 시 폼을 이해할 수 있도록 사용을 지향합니다.
checkbox : 체크박스 생성
button : 버튼 생성
file : 파일 입력 양식 생성
hidden : 보이지 않음
image : 이미지 형태 생성
password : 비밀번호 입력 양식 생성
radio : 라디오 버튼 생성
reset : 초기화 버튼 생성
submit : 제출 버튼 생성
text : 글자 입력 양식 생성
color : 색상 선택 양식 생성
date : 일 선택 양식 생성
datetime : 날짜 선택 양식 생성
datetime-local : 지역 날짜 선택 양식 생성
email : 이메일 입력 양식 생성
month : 월 선택 양식 생성
number : 숫자 생성 양식 생성
range : 범위 선택 양식 생성
search : 검색어 입력 양식 생성
tel : 전화번호 입력 양식 생성
time : 시간 선택 양식 생성
url : URL 주소 입력 양식 생성
week : 주 선택 양식 생성
  1. <label for="idName">이름</label>
  2. <input id="idName" type="text" autocomplete="on" autofocus maxlength="10" disabled />
  3. <input id="idName" type="number" min="0" max="10" />
  4. <input id="idName" type="checkbox" checked />
  5. <input id="idName" type="image" src="button.png" />
select
옵션 메뉴를 제공하는 컨트롤을 나타냅니다.
multiple: 사용자가 두 개 이상의 옵션을 동시에 선택할 수 있음을 명시합니다.
  1. <label for="selectId">select 예시</label>
  2. <select name="selectName" id="selectId">
  3.     <option value="">value값이 null이면 placeholder로 사용</option>
  4.     <option> value="1" selected>select option 1</option>
  5.     <option> value="2" selected>select option 2</option>
  6.     <option> value="3" selected>select option 3</option>
  7. </select>
textarea
여러 줄의 글자를 입력할 때 사용합니다.
cols : textarea의 너비를 지정
rows : textarea의 높이를 지정
textarea 태그 안에서는 줄 바꿈 및 탭을 인식하기 때문에 코드 정렬을 위한 탭, 엔터 사용을 지양합니다.
  1. <label for="example">textarea 예시</label>
  2. <textarea id="example" name="example" rows="5" cols="100"></textarea>
button
type을 지정해주지 않으면 기본값 submit으로 폼이 전송될 수 있으므로 전송 기능이 아닌 경우 type="button"을 선언하여 ui를 제어한다.
  1. <button type="button">일반 버튼</button>
  2. <button type="submit">전송 버튼</button>
radio, checkbox
input과 label을 그룹핑하지 않고 항목 전체를 그룹핑해 불필요한 중첩 구조를 방지합니다.
  1. <div class="radio-wrap">
  2.     <input type="radio" name="radioGroup" id="radioSample1" />
  3.     <label for="radioSample1"><span class="check"></span>Option A</label>
  4.     <input type="radio" name="radioGroup" id="radioSample2" />
  5.     <label for="radioSample2"><span class="check"></span>Option B</label>
  6.     <input type="radio" name="radioGroup" id="radioSample3" />
  7.     <label for="radioSample3"><span class="check"></span>Option C</label>
  8. </div>

Table

table
행과 열로 이루어진 표를 나타냅니다.
caption
표의 설명 또는 제목을 나타냅니다.
스크린리더 사용 시 테이블이 읽히기 전 테이블에 대한 정보 파악이 어려우므로 제공해 줍니다.

  1. <colgroup>
  2.     <col style="width: 100px;">
  3.     <col style="width: 200px;">
  4.     <col style="width: 300px;">
  5. <colgroup>
thead, tbody, tfoot
thead: 표의 머리글을 그룹핑합니다.
tbody: 표의 내용을 그룹핑합니다.
tfoot: 표의 바닥글을 그룹핑합니다.
thead, tfoot 요소는 table내에서 한 번만 사용할 수 있습니다.
th
scope 애트리뷰트로 현재 셀의 어느 셀의 제목인지 명시합니다.

크로스 브라우징

크로스 브라우징이란?
서로 다른 브라우저들끼리 서로 호환이 가능하도록 하는 '웹 페이지간의 상호 호환성'을 의미합니다.
모든 브라우저에서 100% 똑같이 보이도록 만드는 동일성이 아닌 동등성(등가성)을 의미합니다.
어느 한쪽에 최적화되지 않고, 공통 요소를 사용하여 웹 페이지를 제작하는 기법으로 동등한 수준의 정보, 기능 접근에 초점을 맞춥니다.
관련 접두사
-wekit- : 크롬, 사파리 브라우저용
-moz- : 파이어폭스 브라우저용
-ms- : 익스플로러 브라우저용
-o- : 오페라 브라우저용
CSS 기본 설정 (크로스브라우징을 위한 초기화)
브라우저별로 css 기본값이 상이하여, css적용시 다른 결과물을 보여줍니다. 따라서 아래와 같은 reset.css를 작성해줍니다.
블록 타입의 태그 기본값 설정
  1. 블록 타입 속성 ('*' 사용은 지양합니다) {margin:0; padding:0; border:0;}
font, table, input, img 등 css속성값 초기화작업이 필요합니다.
reset.css 파일을 참고 바랍니다.
