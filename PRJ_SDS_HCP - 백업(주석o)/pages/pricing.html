    모든 상품 보기
    Compute
    • Virtual Server 클라우드 컴퓨팅에 최적화 된 가상 서버
    • VM Auto-Scaling 수요에 따라 자원을 자동으로 확장/축소하는 서비스
    • Bare Metal Server 가상화하지 않고 사용하는 고성능 물리 서버
    Storage
    • Block Storage 대규모 데이터 처리와 DB 워크로드에 사용되는 스토리지
    • File Storage 네트워크를 통해 다수의 클라이언트가 파일을 공유하는 스토리지
    • Object Storage 웹 기반 데이터 저장 공간을 제공하는 객체 스토리지
    • Backup 데이터를 백업하고 복구하는 서비스
    Database
    • DB Service 웹 환경에서 Database를 간편하게 생성하고 관리하는 서비스
    Container
    • Kubernetes Engine Kubernetes 컨테이너 오케스트레이션 서비스
    • Kubernetes Apps Kubernetes 클러스터에 즉시 배포할 수 있도록 SW를 이미지화하여 제공하는 서비스
    Networking
    • VPC 독립된 가상 네트워크 공간을 제공하는 서비스
    • Security Group VM/VPC의 트래픽을 제어하는 가상 방화벽
    • Load Balancer 서버 트래픽 부하를 자동으로 분산하는 서비스
    • DNS 간편하게 도메인을 설정하고 관리하는 서비스
    • GSLB 안정적인 네트워크 트래픽을 위한 DNS 기반의 로드 밸런싱
    • VPN 외부 네트워크와 클라우드 네트워크를 연결하는 서비스
    Security
    • DDoS 대응 대용량 네트워크 트래픽 공격 탐지 및 대응 서비스
    • IDS 홈페이지 및 애플리케이션 서버에 대한 침입을 실시간으로 탐지 및 대응하는 서비스
    • WAF 웹 취약점과 공격으로부터 웹 애플리케이션을 보호하는 서비스
    • 웹 취약점 점검 웹 서비스의 보안 취약점 점검 및 대응 방법 제공 서비스
    • 인증서 관리 SSL/TLS 인증서 등록 및 통합 관리 서비스
    Application Service
    • API Gateway API를 손쉽게 관리하고 모니터링하는 서비스
    • Mail/SMS/Push Mail, SMS, Push 알림 기능을 제공하는 서비스
    Analytics
    • Kafka Confluent Platform의 Kubernetes 클러스터를 생성/관리하는 서비스
    AI Service
    • Kubeflow Kubernetes 기반의 오픈소스 Machine Learning 플랫폼
    Management
    • Logging & Audit 사용자의 활동 내역을 수집하고 분석하는 서비스
    • Cloud Monitoring 클라우드 인프라의 안정적인 운영을 지원하는 서비스

상품별 요금제

    • Virtual Server

      • Virtual Server 요금은 무약정 및 약정(1,3년), 사용 OS에 따라 결정됩니다.
      • 무약정 선택시 시간단위의 요금이 적용되고, 1년 또는 3년 약정 선택시 월단위 요금이 적용됩니다.
      • 월단위 요금은 1개월(매월 1일~말일) 기준 이용료이며, 월중 신청/해지시에는 일할 계산됩니다.
      • 약정기간내 해지시 위약금이 발생하고, 해지위약금은 [월요금ｘ50%ｘ약정기간 잔여개월수] 로 산정합니다.
      • 기본 제공되는 CPU는 Intel CascadeLake 이며, OS Disk는 100GB 입니다.
      • 제공되는 OS중 오픈소스는 Community 버전을 제공하며, 기술지원 필요시 옵션내 OS Carepack 참고하시면 됩니다.
        또한 Ubuntu(상용) 및 SLES(SUSE Linux Enterprise Server)는 RHEL 요금, Oracle Linux는 Windows 요금을 적용합니다.
      • 모든 요금은 부가세 미포함 요금입니다.

      • Load Balancer 요금은 Load Balancer Size별 요금과 서비스 수만큼의 요금 합으로 시간단위로 과금합니다.
      • VPC당 Small/Medium은 합산하여 3개까지 생성가능, Large는 1개만 SR로 생성 가능합니다.
        (Small만 3개 사용 또는 Small 2개 + Medium 1개 사용 또는 Medium만 2개 사용 등)
      (단위 : 원/시간, 부가세 별도)
      구분 Load Balancer 요금 서비스 요금 비고
      Small 26 10 서비스 20개까지 (100Mbps)
      Medium 29 서비스 100개까지 (1Gbps)
      Large 35 서비스 1000개까지 (5Gbps)
      서비스란 Load Balancer에서 특정 Virtual Server그룹에 할당한 vIP:Port (예: 10.1.1.1:80)를 말합니다.
      요금 예시)
      서비스 50개, 730 시간 사용시 : 기본요금 Medium 29(원) * 730(시간) + 10(원) * 50(개) * 730(시간) = 386,170원입니다.

    • WAF

      • WAF 요금은 URL別 월 중 최대 트래픽에 따라 결정됩니다.
      • 월단위 요금 (매월 1일~말일) 적용하며, 월중 신청/해지시에는 일할 계산됩니다.
      • 약정 기간은 1년이며, 약정기간내 해지시 위약금이 발생하고, 해지 위약금은 [월요금ｘ75%ｘ약정기간 잔여개월수] 로 산정합니다.
      • 인프라 비용은 별도입니다.
      • 서비스 개시 이후, 약정된 최대 트래픽을 초과할 경우 기존 상품 해지 후 신규 신청이 필요합니다.
      • 트래픽 구간 변경을 위한 해지 후, 1주일 이내에 신규 신청 SR이 접수된다는 전제 하에 위약금은 없습니다.
        단, 서비스 개시 이후 6개월 내 해지시에만 해당합니다. (6개월 이후 해지시에는 신규 신청시에도 위약금 부과됩니다.)
      • 서버 정지와 무관하게 청구됩니다. 과금을 원치 않으시면 서버 정지시 WAF 계약도 함께 해지해 주시기 바랍니다.
      (단위 : 원/월, 부가세 별도)
      트래픽 구간 월 요금 비고
      ~100Mbps 189,000원 1년 약정
      ~300Mbps 300,000원
      ~500Mbps 450,000원
      ~1Gbps 675,000원
      요금 예시) WAF ~100Mbps 구간 상품에 대해 8/15부터 서비스 개시를 가정하면,
      8월 : 8/15~8/31(17일)*6,300원(일할 요금), 9월~차년도 7월 : 189,000원/월, 차년도 8월 : 8/1~8/14(14일)*6,300원(일할 요금) 입니다.

    • DDoS 대응

      • DDoS 대응은 무료로 제공됩니다.
        단, WAF를 신청한 고객에 한하여 제공되며 DDoS 대응 사용 신청은 별도로 진행해 주셔야 합니다.
      • WAF 계약을 해지하실 경우, 별도로 DDoS 계약도 해지해 주시기 바랍니다.

    • IDS

      • IDS 요금은 탐지 정책이 적용된 Virtual Server의 개수에 따라 결정됩니다.
      • 월단위 요금 (매월 1일~말일) 적용하며, 월중 신청시에는 일할 계산됩니다.
      • 약정 기간은 1개월이며, 인프라 비용은 별도입니다.
      • 서버 정지와 무관하게 청구됩니다. 과금을 원치 않으시면 서버 정지 시 IDS 계약도 함께 해지해 주시기 바랍니다.
      (단위 : 원/월, 부가세 별도)
      과금 구분 월 요금 비고
      Virtual Server 70,000원 1개월 약정
      요금 예시) 1개 Virtual Server에 IDS 1개월 적용 신청시, 적용시점을 8/15부터로 가정하면,
      월별 청구 금액은 8월 : 8/15~8/31(17일)*2,334원(일할 요금), 9월 : 9/1~9/14(14일)*2,334원(일할요금) 입니다.

    • 웹 취약점 점검

      • 웹취약점 점검 요금은 점검 횟수에 따라 결정되며, 점검 1회당 URL 1개에 대한 점검이 제공됩니다.

      • 점검 진행율 50% 이상일 경우 점검에 대한 비용이 과금됩니다.
        점검 진행율은 점검을 진행하는 총 웹페이지 수를 기준으로 산정됩니다.

      과금 구분 요금 비고
      진단 횟수 당 400,000원 점검 1회당 URL 1개 점검
