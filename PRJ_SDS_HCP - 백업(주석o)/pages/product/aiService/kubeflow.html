    모든 상품 보기
    Compute
    • Virtual Server 클라우드 컴퓨팅에 최적화 된 가상 서버
    • VM Auto-Scaling 수요에 따라 자원을 자동으로 확장/축소하는 서비스
    • Bare Metal Server 가상화하지 않고 사용하는 고성능 물리 서버
    Storage
    • Block Storage 대규모 데이터 처리와 DB 워크로드에 사용되는 스토리지
    • File Storage 네트워크를 통해 다수의 클라이언트가 파일을 공유하는 스토리지
    • Object Storage 웹 기반 데이터 저장 공간을 제공하는 객체 스토리지
    • Backup 데이터를 백업하고 복구하는 서비스
    Database
    • DB Service 웹 환경에서 Database를 간편하게 생성하고 관리하는 서비스
    Container
    • Kubernetes Engine Kubernetes 컨테이너 오케스트레이션 서비스
    • Kubernetes Apps Kubernetes 클러스터에 즉시 배포할 수 있도록 SW를 이미지화하여 제공하는 서비스
    Networking
    • VPC 독립된 가상 네트워크 공간을 제공하는 서비스
    • Security Group VM/VPC의 트래픽을 제어하는 가상 방화벽
    • Load Balancer 서버 트래픽 부하를 자동으로 분산하는 서비스
    • DNS 간편하게 도메인을 설정하고 관리하는 서비스
    • GSLB 안정적인 네트워크 트래픽을 위한 DNS 기반의 로드 밸런싱
    • VPN 외부 네트워크와 클라우드 네트워크를 연결하는 서비스
    Security
    • DDoS 대응 대용량 네트워크 트래픽 공격 탐지 및 대응 서비스
    • IDS 홈페이지 및 애플리케이션 서버에 대한 침입을 실시간으로 탐지 및 대응하는 서비스
    • WAF 웹 취약점과 공격으로부터 웹 애플리케이션을 보호하는 서비스
    • 웹 취약점 점검 웹 서비스의 보안 취약점 점검 및 대응 방법 제공 서비스
    • 인증서 관리 SSL/TLS 인증서 등록 및 통합 관리 서비스
    Application Service
    • API Gateway API를 손쉽게 관리하고 모니터링하는 서비스
    • Mail/SMS/Push Mail, SMS, Push 알림 기능을 제공하는 서비스
    Analytics
    • Kafka Confluent Platform의 Kubernetes 클러스터를 생성/관리하는 서비스
    AI Service
    • Kubeflow Kubernetes 기반의 오픈소스 Machine Learning 플랫폼
    Management
    • Logging & Audit 사용자의 활동 내역을 수집하고 분석하는 서비스
    • Cloud Monitoring 클라우드 인프라의 안정적인 운영을 지원하는 서비스

Kubeflow

Kubernetes 기반의 오픈소스 Machine Learning 플랫폼
요금 계산기
  • 특징
  • 구성도와 상세기능
  • 요금 기준
  • 연관 상품
Kubeflow는 머신러닝 모델의 개발, 학습, 배포 과정 전체 파이프라인의 반복적인 작업을 자동화하는 오픈소스 Machine Learning 플랫폼입니다. Kubernetes 기반의 MLOps 환경을 제공하며, 학습 데이터와 모델, 운영 데이터의 통합적인 관리가 가능합니다.

특징

  1. Cloud Native MLOps 환경 제공

    Kubeflow 서비스는 클라우드에 최적화 된 머신러닝 모델 개발 환경을 제공하며, Kubernetes 기반으로 다양한 오픈소스와의 연계가 편리합니다.

  2. 빅데이터 사용 편의성

    Tensorflow, Pytorch, Scikit-learn, Keras 등 다양한 머신러닝 프레임워크를 지원하는 표준화된 환경을 제공합니다. 머신러닝 모델의 개발, 학습, 배포 과정의 전체 Pipeline을 자동화하여 제공함으로써 모델 구성 및 생성이 쉽고 재사용이 용이합니다.

  3. Add-on Feature 지속 추가

    GPU 자원을 효율적으로 활용할 수 있는 잡 스케쥴러(FIFO, Bin-packing, Gang기반), GPU 자원 모니터링, Kubeflow 엔진 로깅 등 다양한 Add-on Feature들을 지속적으로 추가하여 제공합니다.

구성도와 상세기능

Kubeflow 서비스 생성
  • 신청 : 사용자가 지정한 Kubernetes 클러스터 대상으로 Kubeflow 자동배포 및 서비스 구성
  • 조회 : 상품 목록, Kubeflow 버전, 자원 현황, Running/Stop 상태 정보 제공
  • 삭제 : 생성한 KubeFlow 모듈 삭제
Kubeflow 서비스 기능
  • 기본 제공 기능
    • Jupyter Notebook 제공 (모델 개발, 학습, 추론)
    • Workflow 자동화 (Machine Learning Pipeline 기반)
  • Samsung Cloud Platform 추가 제공 기능
    • GPU Job Scheduling
    • ML Framework 이미지 (Tensorflow, PyTorch, Mxnet 등)
    • Kubeflow 엔진 모니터링/로깅, 인증 연동

요금 기준

  • Kubeflow 서비스는 Mini, Enterprise 중 선택한 요금과 vCore당 라이선스 요금의 합으로 시간단위 과금합니다.
  • 서비스 구성을 위한 선행상품(K8s Engine, Virtual Server, Storage 등)은 별도 과금합니다.
(단위: 원/시간, 부가세 별도)
