    모든 상품 보기
    Compute
    • Virtual Server 클라우드 컴퓨팅에 최적화 된 가상 서버
    • VM Auto-Scaling 수요에 따라 자원을 자동으로 확장/축소하는 서비스
    • Bare Metal Server 가상화하지 않고 사용하는 고성능 물리 서버
    Storage
    • Block Storage 대규모 데이터 처리와 DB 워크로드에 사용되는 스토리지
    • File Storage 네트워크를 통해 다수의 클라이언트가 파일을 공유하는 스토리지
    • Object Storage 웹 기반 데이터 저장 공간을 제공하는 객체 스토리지
    • Backup 데이터를 백업하고 복구하는 서비스
    Database
    • DB Service 웹 환경에서 Database를 간편하게 생성하고 관리하는 서비스
    Container
    • Kubernetes Engine Kubernetes 컨테이너 오케스트레이션 서비스
    • Kubernetes Apps Kubernetes 클러스터에 즉시 배포할 수 있도록 SW를 이미지화하여 제공하는 서비스
    Networking
    • VPC 독립된 가상 네트워크 공간을 제공하는 서비스
    • Security Group VM/VPC의 트래픽을 제어하는 가상 방화벽
    • Load Balancer 서버 트래픽 부하를 자동으로 분산하는 서비스
    • DNS 간편하게 도메인을 설정하고 관리하는 서비스
    • GSLB 안정적인 네트워크 트래픽을 위한 DNS 기반의 로드 밸런싱
    • VPN 외부 네트워크와 클라우드 네트워크를 연결하는 서비스
    Security
    • DDoS 대응 대용량 네트워크 트래픽 공격 탐지 및 대응 서비스
    • IDS 홈페이지 및 애플리케이션 서버에 대한 침입을 실시간으로 탐지 및 대응하는 서비스
    • WAF 웹 취약점과 공격으로부터 웹 애플리케이션을 보호하는 서비스
    • 웹 취약점 점검 웹 서비스의 보안 취약점 점검 및 대응 방법 제공 서비스
    • 인증서 관리 SSL/TLS 인증서 등록 및 통합 관리 서비스
    Application Service
    • API Gateway API를 손쉽게 관리하고 모니터링하는 서비스
    • Mail/SMS/Push Mail, SMS, Push 알림 기능을 제공하는 서비스
    Analytics
    • Kafka Confluent Platform의 Kubernetes 클러스터를 생성/관리하는 서비스
    AI Service
    • Kubeflow Kubernetes 기반의 오픈소스 Machine Learning 플랫폼
    Management
    • Logging & Audit 사용자의 활동 내역을 수집하고 분석하는 서비스
    • Cloud Monitoring 클라우드 인프라의 안정적인 운영을 지원하는 서비스

아키텍처 센터

아키텍처 센터에서는 레퍼런스 아키텍처 다이어그램, 기술가이드, 아이콘 등을 제공합니다.
클라우드 상품을 활용해 새로운 워크로드를 만들거나 현재 운영 중인 업무를 클라우드로 이전하는데 참고하세요.
상품 아이콘 및 활용 가이드
    • IaaS

      On-premises 고객 네트워크의 클라우드 확장

      On-premises 인프라와 클라우드를 연결하는 네트워크 확장 방안
      IaaS

      GSLB를 활용한 서버 부하 분산

      GSLB를 활용한 클라우드와 On-premises 데이터센터에 동시에 구성된 서비스의 부하 분산
      IaaS

      재해 복구

      Virtual Server, Object Storage 등의 클라우드 자원을 활용한 비용 효율적인 재해복구 시스템 구축

      IaaS

      Object Storage를 활용한 데이터 백업

      저가형 Object Storage를 활용한 Virtual Server의 OS와 DB 백업
      IaaS

      Auto-Scaling을 통한 서버 자동 확장

      Auto-Scaling을 활용한 예측하지 못한 부하 증가에 탄력적 대응
      IaaS

      Standard VPC를 활용한 Virtual Server 기반 DMZ 웹 서비스

      인터넷 통신이 가능한 고객 전용 네트워크와 확장성이 뛰어난 Virtual Server를 활용한 웹서비스 구성

      IaaS

      Dedicated VPC를 활용한 고객 전용 네트워크 구성

      고객 사설 네트워크 대역을 클라우드 자원에서도 할당할 수 있는 구성
      IaaS

      Object Storage를 활용한 멀티 Web Instance간 컨텐츠 공유

      객체 기반 Object Storage와 S3 API를 활용한 멀티 Web Instance간 파일 컨텐츠 공유
      IaaS

      Bare Metal Server를 활용한 고가용성 Oracle DBMS 구성

      클라우드 Bare Metal Server 와 Block Storage를 이용한 물리 서버 기반의 고가용성 Oracle DBMS 구성

      PaaS

      Container기반 Web Application

      Kubernetes를 활용해 자동화 관리/실행이 가능한 Container 기반 어플리케이션 구현
      PaaS

      Microservice with Kubernetes

      클라우드 네이티브 어플리케이션 개발을 위한 Kubernetes Engine, API Gateway, Message Queue 등의 클라우드 서비스 아키텍처

      PaaS

      Cloud기반 MLOps서비스

      Kubernetes기반 MLOps를 지원하는 오픈소스 Kubeflow 구성과 활용
      PaaS

      Container기반 CI/CD Pipeline

      어플리케이션 개발의 편리성과 협업을 지원하는 통합 개발 환경 제공
      PaaS

      Cloud기반 고가용성 DBMS 구성

      클라우드에서 MySQL, PostgreSQL 계열의 DBMS에 대한 고가용성 구성
      PaaS

      대용량 데이터 처리 아키텍처

      분석 전용 Database인 Vertica 클러스터를 클라우드에 쉽고 빠르게 구축 및 관리하는 방안
      PaaS

      Serverless를 활용한 Backend API 개발

      실행 코드만으로 필요한 자원이 동적으로 할당되고 확장되는 Cloud Functions을 활용한 Backend API 개발

      PaaS

      실시간 데이터 Streaming

      데이터 대량 Batch 처리와 실시간 스트리밍 방법을 활용한 대량 데이터 처리 아키텍처
      솔루션 구현

      웹 호스팅

      Virtual Server와 DB Service를 이용해 빠르게 WordPress 기반의 웹사이트 구축 및 호스팅 환경 구성

      솔루션 구현

      On-premises 워크로드의 클라우드 Bursting

      Hybrid Cloud 연결을 기반으로 대규모 시스템 부하에 대응하여 On-premises 리소스를 클라우드로 확장

      마이그레이션

      On-premises NAS의 Object Storage 전환

      On-premises 데이터센터 NAS 스토리지에 보관된 데이터를 클라우드 Object Storage로 전환
      마이그레이션

      On-premises Oracle Data의 클라우드 이행

      Bare Metal Server과 VPN를 활용한 On-premises Oracle 데이터의 클라우드 이행방안
    • IaaS
